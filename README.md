# VisualRegressionTestingSample
DroidKaigi2020: [Androidでもビジュアルリグレッションテストをはじめよう](https://droidkaigi.jp/2020/timetable/156578) のサンプルアプリです

### スライド

https://speakerdeck.com/keidroid/droidkaigi2020-androiddemobiziyuaruriguretusiyontesutowohazimeyou

### 概要

以下の環境で動作確認しています。

- macOS 10.15
- Android Studio 3.5
- デバイス: 3.7inch WVGA(480x800) API26 

reg-suitのコマンドは実行できる前提となっており、別途行う必要があります。
- https://github.com/reg-viz/reg-suit#getting-started
